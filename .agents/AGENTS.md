# AI Agent Rules & Constraints

## Code Modification Rules
- *DO NOT* delete, overwrite, or refactor any existing code unless explicitly instructed by the user.
- If you need to modify existing code, you must only update the specific lines required for the task.
- When adding new features, always append or integrate them into the existing structure without deleting surrounding logic, comments, or helper functions.
- If you think a piece of code is redundant or needs removal, *ASK for user permission first* in the chat before making changes to the file.
- Preservation of existing code is the highest priority.
